Background: I had chatgpt generate a small program to view images for me. I tend to download 'memes' since the cool ones get deleted by mods.
I use this program to view the memes, move them to directories, or just delete them, all from the keyboard.

**memeview** is a minimalist, high-performance image viewer written in C++ using the Qt6 framework. Designed for power users who live in the terminal, it is optimized for the rapid curation, sorting, and archiving of large image collections.

## Features

*   **Zero-Lag Navigation:** Built to cycle through thousands of images instantly.
*   **Vim-Inspired Bindings:** Navigate and manage your library without leaving the home row.
*   **File Management:** Move, rename, or delete files on the fly while reviewing them.
*   **Clipboard Integration:** Copy image data directly to the system clipboard for immediate sharing.
*   **Evolutionary Image Handling:** Automatically handles EXIF orientation and supports modern formats like WebP.

## Installation

### Dependencies
*   **Qt 6.x** (Core, Gui, Widgets)
*   **C++17** compatible compiler (GCC/Clang)

### Build Instructions
1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/memeview.git
    cd memeview
    ```
2.  Generate the Makefile and compile:
    ```bash
    qmake6
    make
    ```
3.  Install to `~/bin`:
    ```bash
    make install
    ```

## Usage
Launch the viewer by passing one or more image files, or use wildcards for entire directories:
```bash
memeview ~/Downloads/*.webp
```
